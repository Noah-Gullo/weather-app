# Weather App
## Link:
https://noah-gullo.github.io/weather-app/

## Description:
A weather applciation that allows you to search for current weather information at a given location including temperature, humidity, current condition, wind speed and more.

## Skills:
This provided good practice writing asynchronous functions and dealing with promises along with async/await while interfacing with Visual Crossing's Weather API.

## Reflection:
This was a really fun project. It was gratifying leveraging an API to serve data and then chose how I want to display certain bits of data. It felt rewarding being able to display information sourced from other people and I can definitely appreciate the power of APIs.
